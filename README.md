## Getting Started

### Install with Yarn
Install ESLint config Cecilia from Yarn.
```
yarn add eslint-config-cecilia --dev
```

### Usage
Add `.eslintrc.js` on your project root directory.
```
module.exports = {
    'extends': 'cecilia'
};
```
